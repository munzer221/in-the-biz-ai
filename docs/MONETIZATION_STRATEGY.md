### **AI Cost Optimization - Smart Routing**

**Problem:** Not all queries need expensive AI models.

#### **Implementation:**
```dart
String selectAIModel(String userMessage) {
  // Simple database queries
  if (isSimpleQuery(userMessage)) {
    return "gemini-2.0-flash-lite"; // $0.075/1M input (10x cheaper)
  }
  
  // Complex reasoning, advice, image analysis
  return "gemini-3-flash-preview"; // $0.50/1M input (smart AI)
}

bool isSimpleQuery(String message) {
  final simplePatterns = [
    "how much", "what was", "show me", "total",
    "last week", "last month", "yesterday",
  ];
  
  return simplePatterns.any((pattern) => 
    message.toLowerCase().contains(pattern)
  );
}
```

**Savings:** 60% of queries use cheap AI = 10x cost reduction on those queries
